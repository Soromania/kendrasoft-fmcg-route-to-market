## Project Overview
The most advanced engagement in the KENDRASOFT portfolio to date — 
a comprehensive route-to-market intelligence solution for an FMCG 
beverages business, featuring a 7-table star schema, advanced T-SQL 
analytical functions, Power BI Time Intelligence reporting, and a 
fully automated PowerPoint deck generated programmatically via pptxgenjs.

## Business Problem
The client needed a data-driven framework to prioritise distributors, 
optimise trade channels, and plan territories — moving beyond gut-feel 
decisions toward evidence-based route-to-market strategy.

## Solution Delivered

### Database Architecture
- Designed a **7-table star schema** in Microsoft SQL Server
- Tables: Routes, Distributors, Trade Channels, SKUs, Promotions, 
  Sales Force, Transactions
- Captured the complete FMCG route-to-market architecture

### Advanced T-SQL Analytics
- **NTILE()** — distributor tiering and segmentation
- **PERCENTILE_CONT()** — rep efficiency banding
- **Recursive CTEs** — hierarchical route and territory analysis
- **PIVOT** — promotional ROI cross-tabulation
- Multi-dimensional performance scoring across routes and channels

### Power BI Time Intelligence Report
- **SAMEPERIODLASTYEAR** — year-on-year performance comparisons
- **TOTALYTD** — cumulative year-to-date tracking
- **Rolling 3-month averages** — trend smoothing and forecasting signals
- Multi-page report with dedicated views per business dimension

### Automated PowerPoint Deck
- Programmatically generated branded executive presentation using **pptxgenjs**
- Data-driven slide content produced automatically from the analytics pipeline
- Delivered a professional, client-ready PowerPoint deck without manual design

## Key Outcomes
- Enabled data-driven distributor prioritisation and territory planning
- Delivered automated reporting pipeline reducing manual presentation effort
- Provided a scalable route-to-market intelligence framework reusable across FMCG clients
